estudo fácil
visão geral do projeto

o estudo fácil é uma aplicação web criada para funcionar como uma central de organização acadêmica. o sistema foi desenvolvido para ajudar estudantes a controlar suas atividades, provas, prioridades e rotina de estudos, reduzindo a desorganização e facilitando o acompanhamento de prazos. ele foi construído utilizando o framework flask e segue o padrão de arquitetura mvc, separando os dados, as rotas e a interface para tornar o projeto mais organizado e compreensível.

funcionalidades

o sistema permite o registro de atividades e provas em um único local, organiza e exibe os prazos de entrega, possibilita a criação de um plano diário de estudos e oferece filtros por matérias para facilitar a navegação e a localização de tarefas específicas.

tecnologias utilizadas

o backend utiliza python 3 e o framework flask, responsável pelo roteamento. o banco de dados utilizado é sqlite por meio do flask sqlalchemy, que realiza a persistência das informações. no frontend são usados html e css para estruturar e apresentar a interface ao usuário.

como instalar e executar

para rodar o estudo fácil no computador é necessário ter python 3 e git instalados. depois disso, basta clonar o repositório, acessar a pasta do projeto e criar um ambiente virtual para isolar as dependências. após ativar o ambiente virtual, é preciso instalar as dependências do arquivo requirements.txt. com isso feito, o sistema pode ser iniciado com o comando para executar o arquivo principal e estará disponível no navegador no endereço padrão local.

reflexão crítica sobre o sistema

o estudo fácil foi desenvolvido para resolver o problema comum da desorganização acadêmica. estudantes lidam diariamente com diversas disciplinas, atividades, datas de entrega e provas. quando essas informações não estão organizadas em um único lugar, surge a sensação de sobrecarga, esquecimento de tarefas, acúmulo de atividades e estresse. alunos que trabalham, estudam em cursos com grande quantidade de matérias ou têm pouca prática com autogestão sentem esse problema de forma ainda mais intensa.

um sistema web foi escolhido por ser acessível em qualquer dispositivo com internet e por permitir que o aluno atualize seu planejamento a qualquer momento. além disso, uma plataforma digital pode centralizar e organizar automaticamente informações de estudo, algo que ferramentas analógicas como agendas, planners ou quadros brancos não fazem sem esforço manual constante. outras soluções digitais, como aplicativos de notas ou calendários, não são específicas para a estrutura acadêmica, enquanto o estudo fácil foi pensado diretamente para a rotina estudantil.

mesmo oferecendo várias vantagens, o sistema também tem limitações. ele não resolve a questão da motivação e disciplina pessoal, pois o aluno ainda precisa manter seus dados atualizados e seguir o planejamento criado. também existe a barreira da exclusão digital, já que alunos sem acesso constante à internet podem ter dificuldade em utilizar a ferramenta. além disso, o sistema precisa tratar com cuidado a questão da privacidade, garantindo a proteção dos dados do estudante e o uso seguro das informações.

durante o desenvolvimento, aprendemos a aplicar o padrão mvc dentro do flask, separando modelos, rotas e templates. o modelo foi responsável pela estrutura e armazenamento dos dados; as rotas conectaram a lógica do sistema com a interface; os templates organizaram a apresentação visual das informações ao usuário. além dos aprendizados técnicos, o projeto mostrou a importância de observar o problema real do estudante. percebemos que um software útil não precisa ser complexo, mas sim resolver de forma clara e objetiva uma dor específica do usuário. o estudo fácil nasceu dessa ideia: ser uma ferramenta simples, prática e diretamente voltada para melhorar a rotina acadêmica.
